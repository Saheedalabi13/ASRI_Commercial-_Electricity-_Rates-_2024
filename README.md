# ASRI_Commercial-_Electricity-_Rates-_2024

## Project Overview

This project investigates the impact of utility ownership type on commercial electricity rates across the United States.

## Research Question

Does utility ownership type significantly influence commercial electricity rates across the United States?

## Hypotheses

- H₀: There is no significant difference in commercial electricity rates among utility ownership types.
- H₁: At least one utility ownership type has a significantly different commercial electricity rate.

 ## Dataset

- Electricity Utility Look-up by ZIP Code (2024)
- United States commercial electricity data

## Methodology

- Data Cleaning and Preprocessing
- Descriptive Statistics
- Data Visualization
- One-Way ANOVA
- Tukey HSD Post-hoc Analysis

## Key Findings

- Significant differences were found in commercial electricity rates across ownership types.
- ANOVA: F(6, 77006) = 268.58, p < 0.001.
- Most pairwise ownership comparisons were significantly different according to Tukey HSD.
