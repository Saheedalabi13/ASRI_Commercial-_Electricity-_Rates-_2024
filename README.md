# ASRI_Commercial-_Electricity-_Rates-_2024

## Project Overview

This project investigates the impact of utility ownership type on commercial electricity rates across the United States.

## Research Question

Does utility ownership type significantly influence commercial electricity rates across the United States?

## Hypotheses

- H₀: There is no significant difference in commercial electricity rates among utility ownership types.
- H₁: At least one utility ownership type has a significantly different commercial electricity rate.
